# Getting Help

"man" command:
```
man <tool>
```
For example:
```
man curl
```

For quicker glance:
```
<tool> --help
```

For example:
```
curl --help
```

Or a short version:
```
<tool> -h
```

```
curl -h
```

Search manuals for a given keyword:
```
apropos <keyword
```

```
apropos sudo
```
